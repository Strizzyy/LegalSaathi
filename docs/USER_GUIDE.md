## Troubleshooting

### Common Issues

#### Document Upload Problems
**Issue**: File won't upload
**Solutions**:
- Check file size (max 20MB)
- Verify file format (PDF, DOC, DOCX, TXT, images)
- Try different browser
- Clear browser cache

#### Analysis Errors
**Issue**: Analysis fails or takes too long
**Solutions**:
- Ensure document has readable text
- Try smaller file size
- Check internet connection
- Refresh page and retry

#### Translation Issues
**Issue**: Translation not working
**Solutions**:
- Verify language is supported
- Check document was analyzed first
- Try different target language
- Contact support if persistent

#### Voice Features Not Working
**Issue**: Microphone or audio not functioning
**Solutions**:
- Grant microphone permissions
- Check browser audio settings
- Try different browser
- Ensure device has microphone/speakers
